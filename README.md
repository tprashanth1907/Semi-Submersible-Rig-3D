## 📋 About the Project

Semi-Submersible-Rig-3D lets you view a detailed 3D model of a semi-submersible oil rig right in your web browser. This project uses interactive 3D visualization so you can explore the rig from different angles. It’s designed for people interested in offshore engineering, oil and gas platforms, or those curious about complex offshore structures.

The 3D model is based on mesh files commonly used in engineering, such as `.obj`, `.mtl`, and `.ply`. The rig’s parts are accurately represented, allowing you to see the structure clearly. You do not need any technical skills to use this application. It runs on Windows through your web browser without installing special software.
